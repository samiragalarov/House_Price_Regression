# House Price Prediction - Practice Assignment

## Overview
This is a test 
This assignment is designed to consolidate your understanding of data preprocessing, feature engineering, and exploratory data analysis (EDA) by applying these techniques to a real-world dataset from the Kaggle House Prices competition.

## Learning Objectives

By completing this assignment, you will:

1. Apply data cleaning techniques to handle missing values, duplicates, and outliers
2. Perform comprehensive data preprocessing including encoding and scaling
3. Engineer new features to improve predictive power
4. Conduct thorough exploratory data analysis to understand data patterns
5. Prepare a clean, analysis-ready dataset for future modeling

## Dataset Description

The House Prices dataset contains information about residential homes in Ames, Iowa. Your task is to prepare this data for predicting house sale prices.

**Files provided:**
- `train.csv` - Training dataset with SalePrice (target variable)
- `test.csv` - Test dataset without SalePrice (for future predictions)
- `data_description.txt` - Detailed description of all features
- `sample_submission.csv` - Example submission format

## Assignment Structure

This assignment is divided into four sequential notebooks. Complete them in order:

### Phase 1: Data Understanding and Cleaning
**Notebook:** `1-Data-Cleaning.ipynb`

Focus on understanding the dataset structure and cleaning the data.

### Phase 2: Data Preprocessing
**Notebook:** `2-Data-Preprocessing.ipynb`

Transform and prepare the data for analysis and modeling.

### Phase 3: Feature Engineering
**Notebook:** `3-Feature-Engineering.ipynb`

Create new features and transform existing ones to improve predictive power.

### Phase 4: Exploratory Data Analysis
**Notebook:** `4-Exploratory-Data-Analysis.ipynb`

Analyze patterns, relationships, and distributions in the data.
